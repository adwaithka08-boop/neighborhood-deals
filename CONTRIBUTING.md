# Contributing to Neighborhood Deals

Thank you for considering contributing to our project! Here are some guidelines to help you get started.

## How to Contribute
1. **Fork the repository**: Click the 'Fork' button at the top right of the page.
2. **Clone the forked repository**: Use `git clone <your-fork-url>` to get a local copy.
3. **Create a new branch**: Use `git checkout -b <your-branch-name>`.
4. **Make your changes**: Work on your feature or bug fix.
5. **Commit your changes**: Use `git commit -m "Your commit message"` to save your changes, e.g. `git commit -m "Add new feature"`.
6. **Push your changes**: Use `git push origin <your-branch-name>`.
7. **Create a pull request**: Go to the original repository and click the 'New Pull Request' button.

## Guidelines
- Write clear, concise commit messages.
- Follow the project's existing code style.
- Add tests for new functionality.

## Code of Conduct
Please note that this project is [protected by a Code of Conduct](CODE_OF_CONDUCT.md), and we expect all contributors to adhere to it.
