# Neighborhood Deals

This repository contains the code for the Neighborhood Deals application.

## Project Structure

- **frontend/** - Contains the frontend code
  - **src/** - Source files for the frontend
  - **public/** - Public assets
  - **package.json** - Frontend dependencies and scripts

- **backend/** - Contains the backend code
  - **src/** - Source files for the backend
  - **config/** - Configuration files for backend setup
  - **package.json** - Backend dependencies and scripts

- **.gitignore** - Git ignore rules
- **docker-compose.yml** - Docker setup for the project
- **Dockerfile** - Dockerfile for building the application

## Installation

Follow the instructions below to get the development environment set up.

### Backend
1. Navigate to the backend folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the backend server:
   ```bash
   npm start
   ```

### Frontend
1. Navigate to the frontend folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the frontend server:
   ```bash
   npm start
   ```

### Docker Setup

To run the application with Docker, execute:
```bash
docker-compose up
```
